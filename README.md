# Amazon Project (HTML & CSS)

A simple static website replicating Amazon's homepage layout using only HTML and CSS.  
The project focuses on basic web design, layout structuring, and styling concepts without using JavaScript.

## Features
- Amazon-like homepage UI
- Header with navigation bar
- Product grid layout
- Footer section with links

## Technologies Used
- HTML
- CSS
